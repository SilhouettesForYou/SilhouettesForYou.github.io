GitKraken 简介

GitKraken 是一款界面十分美观的 Git 可视化管理工具，比较遗憾的是 GitKraken 是收费软件，类似的免费软件有 Sourcetree ，不过使用体验不如 GitKraken。

GitKraken 连接 GitLab

Start a hosted repo

通过 ** File/Preferences 进入偏好设置界面，选择 Authentication 中的 GitLab Self-Managed ** 页面，输入 GitLab 地址和 Token

获得 Personal Access Token
  1. 进入 ** User Settings，选择 Access Token**
  2. 输入一个 Token 的名字
  3. 勾选 api 和read_user
  4. 生成 Token
  5. 复制 Token（这个 Token 离开页面后将不被保存）
连接成功
