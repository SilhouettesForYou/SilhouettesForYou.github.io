破解 GitKraken

Last updated on 2024-07-29T22:23:06+08:00

GitKraken 简介

GitKraken 是一款界面十分美观的 Git 可视化管理工具，比较遗憾的是 GitKraken 是收费软件，类似的免费软件有 Sourcetree ，不过使用体验不如 GitKraken。

GitKraken 连接 GitLab

Start a hosted repo

通过 File/Preferences 进入偏好设置界面，选择 Authentication 中的 GitLab Self-Managed 页面，输入 GitLab 地址和 Token

获得 Personal Access Token
  1. 进入User Settings，选择Access Token
  2. 输入一个 Token 的名字
  3. 勾选 api 和read_user
  4. 生成 Token
  5. 复制 Token（这个 Token 离开页面后将不被保存）
连接成功

破解

安装yarn

运行破解项目

破解前先确保 GitKraken 已关闭退出，破解工具在 Github 上是一个开源项目 5cr1pt/GitCracken ，将该项目 clone 到本地，使用方法也可参考项目的 README.md 文件，另外也在 Gitee 上也有。仓库拷贝到本地后，进入仓库的 GitCracken/GitCracken 目录，依次执行以下指令运行破解

运行 yarn install 可能会出现连接超时的错误

直接安装对应的 module 即可
