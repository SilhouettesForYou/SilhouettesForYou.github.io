### 强制覆盖本地代码（与git远程仓库保持一致）

```sh
git fetch --all # 拉取所有更新，不同步
git reset --hard origin/master # 本地代码同步线上最新版本（会覆盖本地所有与远程仓库上同名的文件）
git pull # 再更新一次
```
